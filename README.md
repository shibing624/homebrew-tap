# shibing624/homebrew-tap

Homebrew tap maintained by [@shibing624](https://github.com/shibing624).

## Available Formulae

### `treesearch`

Structure-aware document search CLI built with Rust. Indexes files into a local
SQLite FTS5 database and searches by document structure instead of chunked text.

Project: <https://github.com/shibing624/TreeSearch>

```bash
brew tap shibing624/tap
brew install treesearch
ts --help
```

The `treesearch` formula in this tap is generated and pushed automatically by
[`cargo-dist`](https://github.com/axodotdev/cargo-dist) on every TreeSearch
release tag.
